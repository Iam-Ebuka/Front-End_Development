HTML

HTML means Hypertext Markup Language. It is the official language for writting webpages. Html was created in the late 20th century and it has since then evolved in its nature.

There are different versions of HTML. Today we are using HTML5 version which has been made to be more effective. Today, most browsers have met up with the trend as they are now supporting HTML5. What makes a document recognised as a HTML5 document is shown below:

<!DOCTYPE html>

HTML a Programming Language?

This is a question that has come up several times in different forums. Before we can answer this question, We have to define what a programming language is.

A programming language is a set of instructions which tells the computer what to do. HTML is not regarded as a programming language because it doesn't tell the computer to carry out tasks. HTML only gives a webpage a structure.

<header>...</header>
<main>...</main>

CSS

CSS means Cascading Style Sheet. It is used for giving webpages a design. CSS works hand in hand with the HTML.

It picks a HTML element and adds style to it.

body {font-size: 20px; color: white;} * {box-sizing: border-box;}

CSS a Programming Language?

CSS is not a programming language because it doesn't instruct the computer with tasks to accomplish

There are also different versions of CSS with CSS3 being the latest
